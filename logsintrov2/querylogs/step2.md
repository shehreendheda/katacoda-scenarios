With the <a href="https://docs.datadoghq.com/logs/explorer/#simple-groups" target="_blank">Simple Group</a> aggregation, all logs matching the query filter are aggregated into groups based on the value of a log facet. For these groups, you can extract counts of logs per group, unique count of coded values for a facet per group, and statsitical operations on numerical values of a facet per group.
